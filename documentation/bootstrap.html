void initializeControllers(<array> controllers, <array> models, <class> mobiusProcessingStack)
    Initializes all controllers with the set of models. The models are injected onto the controller object, so that from the controller object operations can be performed on them.
    Parameters:
      controllers - all controller classes.
      models - all model classes.
